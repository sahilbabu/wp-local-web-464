Ultimate Image Scraping & Gallery Plugin allows you search and create custom image gallery from google. 

== Description ==
A wordpress plugin to scrap images directly from Google search engine. It comes with an option to insert images directly into the post or create a separate custom gallery.

Features:-

1. Easy to use wp plugin.
2. Compatible with all versions of wordpress.
3. Responsive gallery.
4. Search images directly from WP Admin.
4. Automatically Insert images into posts or create custom gallery.

== Installation ==
1. Unzip the image-scraping-and-gallery.zip
2. Copy image-scraping-and-gallery folder to wp-content/plugins
3. Go to Plugins/Installed Plugins, find Google Images Search And Insert and click Active
4. Enjoy!

== Frequently Asked Questions ==
1. How to use this plugin?
    After installing & activating plugin go in into post and page and at top left corner you will get option "Serch images on google".
