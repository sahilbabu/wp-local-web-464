Delete Duplicate Post, Page, Attachment and Custom Post

== Description ==

Delete Duplicate Post, Page, Attachment and Custom Post by just one Click.

You can delete following duplicate data.

* Post
* Page 
* Custom Post 
* Attachment
* Category
* Taxonomy
* Custom Category

You can delete duplicate data Permanently or move to trash.


In Future We Will Add Schedule Deletion that can be very usefully with auto blogging. 

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
